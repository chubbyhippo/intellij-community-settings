== IdeaVim

The bundled `.ideavimrc` brings the Emacs `init.el` Meow editing model into the
IDE: selection-first editing in normal mode, plus a leader (`<Space>`, also
`Alt-m`) that mirrors Meow's keypad over the Emacs keymap — `SPC x f` = `C-x C-f`
= find-file, `SPC c f` = `C-c f`, `SPC m x` = `M-x`, and so on. See
link:ideavim/README.adoc[the IdeaVim README] for the full key reference.
